Este projeto se refere a um app WEB para pc mas tabém e customizado para abrir em celulares androis ou IOs, onde a principal funcionalidade é traçar um trajeto de viagem de um ponto ao outro, ou vários postos, e mostrar uma animação entre eles de um carro ou moto ou gps, e podendo fazer uma parada de 3s em alguns pontos de interesse do usuário para que assim crie uma animação do trajeto e possa exportar esta animação em um stream de vídeo ou etc...
